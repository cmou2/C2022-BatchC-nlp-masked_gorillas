Introduction

Professional therapy can be shameful, expensive, time consuming, and difficult to acquire. Thus we made The NLP Therapy Chatbot to solve these problems.

  • Fast Responses

    From a user input, a response will be generated, giving answers in minuets instead of hours

  • Accessibility

    The NLP Therapy bot requires no commuting - you can have therapy from the comfort of your home.

  • Free of Cost

    The NLP Therapy Bot is completely free.

  • No Interaction Required

    With inputs being passed to an AI instead of a person, the user can feel comfortable talking about whatever they need.

Pipeline

Pipeline

  • Data Collection

    Data were gathered from several places including github, mental health forums, and public datasets.

  • Data Processing

    The data were then processed, removing a lot of noise and unnecessary information.

  • Training

    The model was trained with GPT-2 using the data that had previously been processed.

  • Iteration

    The data were reprocessed and cleaned further to get a better dataset. Afterwards, the model was retrained using GPT-3 for better results.

  • Website Development

    After completing our model, we created this website in order to make our NLP Therapy Chatbot model available to everyone.

Data

Data

Data Scarcity Issues

Conversations between conselors and patients are private, leading to data scarcity.

Data from CounselChat

Our initial dataset was found on a github repo containing data from counselchat.com, a mental health forum. However, we found that this repo didn't have enough data for our project.

Scraping from MentalHealthForum.net

To fix our aforementioned data scarcity issue, we scraped posts from a public forum called mentalhealthforum.com.

Github/Council Processing

For processing the data from Github/Council, we had to fix formating issues and remove extra languages.

Webscraping Processing

The scraped data from MentalHealthForum needed to be cleaned: several items were removed, including answers that were too short.

Combining the Datasets

The datasets were combined and further cleaned.

Modeling

Modeling

What is GPT-3?

GTP-3 is a neural network made by Open-AI capable of generating text

  • Created by Open-AI
  • GPT-3 is a machine learning model that takes in human language and can interpret the sentences

Why are we using GPT-3?

GPT-3 is a cutting edge artificial intelligent model that we used to train our model. However, it is an expensive model and had long training times, but it was worth the quality.

Training GPT-3

Using the cleaned data from Github, CounselChat, and MentalHealthForum we trained our GPT-3 model. Training a text extension neural network means slowly refining the connections between neurons of the network so that it becomes more and more accurate.
